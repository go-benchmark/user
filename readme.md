# FS Benchmark User 

FS Benchmark User is simulation of users of FS Benchmark

## Installation

Use the go module to install FS Benchmark User.

```go
go get github.com/go-benchmark/user
```

## Usage

```go
import (
    "github.com/go-benchmark/user"
)

u := service.NewUser(opts)

```
